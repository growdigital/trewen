Ysgol Trewen WordPress intro

To login, go to ysgoltrewen.uk/wp/wp-admin/. There is a user called editor for editing content.

The Ysgol Trewen website is powered by WordPress, the world’s most popular, free, website software. See their website for learning how to get started with WordPress.

There are currently three pages on the Ysgol Trewen site: home, dates and staff.

The home and dates page use the same format: each Image and text row is an image, caption & text. At the bottom of the content is a button to add another row.

To insert a photo, upload it first to the Media Library in the highest resolution at 16:9 aspect ratio, and then select it from the Media Library via the Add Image button.

The staff page has a row for each member of staff, with the name, title and photo.

You can edit Global content for Term dates, School details and Footer. Global content is used in more than one place.

To handle multilingual content, the Ysgol Trewen site uses the WPML plugin which has been donated by WPML to the school.
